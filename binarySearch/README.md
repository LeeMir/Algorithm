# Binary Search (이분 탐색)

**이분 탐색이란?** 탐색 범위를 절반씩 줄여가면서 최적해를 구하는 방법

> 1. 탐색에 대한 성공/실패 여부를 반환하는 함수 fn()을 작성
> 1. main에서는 start, end, mid를 선언한 후 반복문 작성
>    - `if (fn(mid))` => `end = mid - 1`
>    - `if (!fn(mid))` => `start = mid + 1`
